This is first task of ELEVATE LABS given to us on 23rd July,2026.

<h1 align="center">Saanya - Responsive Landing Page</h1>

<p align="center">
A modern, clean and fully responsive Landing Page built using HTML5, CSS3 and Vanilla JavaScript.
</p>

<hr>

<h2>Features</h2>

<ul>
  <li>Responsive Design (Desktop, Tablet & Mobile)</li>
  <li>Sticky Navigation Bar</li>
  <li>Custom "Saanya" Logo</li>
  <li>Smooth Scrolling Navigation</li>
  <li>Hero Section with Call-to-Action Button</li>
  <li>About Section</li>
  <li>Services Section</li>
  <li>Modern Contact Section</li>
  <li>Footer with Social Media Links</li>
  <li>Hover Animations</li>
  <li>Responsive Layout using Flexbox & Grid</li>
  <li>Custom Cursor</li>
  <li>Fully Responsive Design</li>
</ul>

<h2>How to Run This Project Locally</h2>

<ol>
<li>Clone or Download this repository.</li>
</ol>

<pre>
git clone https://github.com/SaanyaPandey/Task1-ElevateLabs.git
</pre>

<ol start="2">
<li>Open the project in Visual Studio Code.</li>
<li>Install the <b>Live Server</b> extension.</li>
<li>Open <b>index.html</b>.</li>
<li>Right Click → <b>Open with Live Server</b>.</li>
<li>The landing page will open automatically in your browser.</li>
</ol>

<p><b>Alternative:</b> Simply double-click <code>index.html</code> or open it using Google Chrome, Microsoft Edge, or Firefox.</p>
